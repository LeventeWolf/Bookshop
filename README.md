## Setup & Run

### Prerequisites:

- NodeJS version: 16
- Oracle (database)
- Web browser: Chrome, Mozilla firefox, 

### Install

To install the dependencies go to /client & /server then run: <br>

`npm install`


### Run

#### Webserver

/server -> `npm start`

#### Client

/client -> `npm start`
